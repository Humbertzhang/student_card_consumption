# 学生卡消费账单-后端（STDCC）

## 消费账单接口
http://console.ccnu.edu.cn/ecard/getTrans?userId=Int&days=Int&startNum=Int&num=Int
例：
+ userid:学生学号
+ days:查询的最近天数
+ starNum:开始查询的条数
+ num:结束查询的条数

## 学生卡号的范围
+ 13级:2013210001~2013214857
+ 14级:2014210001~2014214840
+ 15级:2015210001~2015214594
+ 16级:2016210001~2016214642
